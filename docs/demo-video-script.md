# Demo Video Script

## Target length

5 to 7 minutes.

## 1. Intro

“Hello, this is OutcomeIQ, an outcome-aware AI FinOps platform. It helps teams understand not only how much an AI workflow costs, but what it costs per successful business outcome.”

## 2. Problem

“Most tools show token usage, latency or API cost. But if an AI support workflow spends money and fails to resolve a ticket, that cost is failure waste. OutcomeIQ connects workflow cost with verified outcomes.”

## 3. Solution

“The MVP focuses on AI customer-support workflows. It tracks simulated model calls, tool calls, workflow runs, cost calculation, Outcome Contracts, dashboard analytics and recommendations.”

## 4. Login/Register

“I’ll register or login to the local app. This is a protected demo environment using local authentication.”

## 5. Create Project

“Next, I create an organization and project. This project becomes the boundary for workflows, costs, outcomes and recommendations.”

## 6. Run Demo Data

“Now I run the simulated support-ticket scenario. The example ticket is: ‘My payment failed but money was deducted.’ The app creates workflow telemetry, model calls, tool calls, costs and outcomes using a simulated provider.”

## 7. Dashboard

“On the dashboard, we can see total workflow runs, total cost, success rate and cost per successful outcome. This is the core metric. A workflow may be cheap per request but costly per successful ticket if failures are high.”

## 8. Recommendations

“Finally, I generate recommendations. These are simple rule-based recommendations backed by the recorded metrics, not autonomous decisions.”

## 9. Closing

“OutcomeIQ proves that AI FinOps should be outcome-aware. The MVP does not use real AI APIs or cloud billing integration yet, but the architecture is ready for those future extensions.”
